# Curso em Vídeo - Java

Repositório criado para armazenar os exercícios e práticas desenvolvidos durante meus estudos iniciais em Java pelo **Curso em Vídeo**.

Este projeto representa minha primeira fase de aprendizado com a linguagem Java, onde pratiquei conceitos fundamentais de programação, lógica, orientação a objetos e criação de pequenos programas no terminal.

## Objetivo

O objetivo deste repositório é registrar minha evolução nos primeiros estudos com Java, servindo como histórico de aprendizado e base para comparar minha evolução com projetos mais recentes.

## Conteúdos praticados

Durante os exercícios, foram trabalhados conceitos como:

- Sintaxe básica da linguagem Java
- Variáveis e tipos de dados
- Entrada e saída de dados
- Estruturas condicionais
- Estruturas de repetição
- Métodos
- Introdução à Programação Orientada a Objetos
- Criação de classes e objetos
- Prática com pequenos problemas de lógica
